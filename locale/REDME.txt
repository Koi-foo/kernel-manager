[en_US]
=======
Poedit is used for translation.
This program can be installed from the repositories.

Translation.
File draft "kernel_manager.pot"
Open the kernel_manager.pot file in Poedit.
Click "create new translation".
Choose your language.
Translate the contents of the file and save.
2 files * .mo, * .po will appear.
Add them to git: https://github.com/Koi-foo/kernel-manager
Or send by mail: "eg.evgeniy@gmail.com". Post subject: Translation of the kernel manager.

Updating translations.
Open the * .po file with your translation using Poedit.
At the top, click the Catalog menu item and select Update from *.pot file. Select a file.
Complete the translation and save.
Send the new * .po and * .mo translation files to the developer.

Editing or correcting a translation
To fix the translation, open the kernel_manager.po file in the desired language directory.
For example:
locale/ru/LC_MESSAGES/kernel_manager.po
Find and fix the lines you want, then save the file.
After saving the kernel_manager.po file, the kernel_manager.mo file is automatically updated.
Send the updated files to the developer.

[ru_RU]
=======
Poedit используется для перевода.
Эту программу можно установить из репозиториев.

Перевод.
Черновик файла "kernel_manager.pot"
Откройте файл kernel_manager.pot в Poedit.
Нажмите «создать новый перевод».
Выберите свой язык.
Переведите содержимое файла и сохраните.
Появятся 2 файла * .mo, * .po.
Добавьте их в git: https://github.com/Koi-foo/kernel-manager
Или отправьте по почте: "eg.evgeniy@gmail.com". Тема сообщения: Перевод kernel manager.

Обновление перевода.
Откройте * .po файл с вашим переводом с помощью Poedit.
Вверху щелкните пункт меню «Каталог» и выберите «Обновить из файла *.pot». Выберите файл.
Завершите перевод и сохраните.
Отправьте разработчику новые файлы перевода * .po и * .mo.

Редактирование или исправление перевода.
Чтобы исправить перевод, откройте файл kernel_manager.po в каталоге нужного языка.
Например:
locale/ru/LC_MESSAGES/kernel_manager.po
Найдите и исправьте нужные строки, затем сохраните файл.
После сохранения файла kernel_manager.po файл kernel_manager.mo автоматически обновляется.
Отправьте обновленные файлы разработчику.
